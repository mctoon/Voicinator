## Top Discoveries

### 🏆 Best Open-Source Stack for Mac M2

**Recommended Pipeline:**
1. **WhisperX** (or whisper-diarization) for transcription + diarization
2. **Pyannote.audio 3.1** for speaker segmentation
3. **SpeechBrain** or **Resemblyzer** for voice embeddings

### 💡 Key Insight: The Pipeline

All solutions follow this architecture:
```
Audio → VAD → ASR (Whisper) → Speaker Embeddings → Clustering → Labeled Transcript
```

### 🎯 Voice Fingerprinting = Speaker Embeddings

- **x-vectors** or **ECAPA-TDNN** embeddings create unique voice signatures
- These are high-dimensional vectors (512-1920 dimensions)
- Similarity comparison via cosine distance
- Can identify speakers across different recordings

## Critical Findings

### 1. Speaker Diarization vs. Speaker Identification

- **Diarization:** "Who spoke when" (Speaker A, B, C...)
- **Identification:** "This is John Doe speaking" (requires enrollment)
- **Verification:** "Is this John Doe?" (1:1 comparison)

Most open-source tools do diarization. True identification requires voice enrollment database.

### 2. The Enrollment Problem

For true voice fingerprinting (identifying specific people), you need:
- Sample recordings of each speaker
- Enrollment phase to create voice profiles
- Comparison against known profiles

**No enrollment = anonymous speaker labels only (Speaker 1, Speaker 2)**

### 3. Accuracy Factors

| Factor | Impact |
|--------|--------|
| Audio quality | High impact |
| Microphone distance | High impact |
| Background noise | Moderate impact |
| Overlapping speech | High challenge |
| Speaker similarity | Moderate challenge |
| Accent/Dialect | Varies by model |
