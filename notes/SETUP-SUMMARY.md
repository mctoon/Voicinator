## 🔬 Day 1 Research Findings

### The Solution Stack (For Mac M2 32GB)

**Best Open-Source Pipeline:**
1. **WhisperX** or **whisper-diarization** - Transcription + diarization
2. **Pyannote.audio 3.1** - Speaker identification
3. **SpeechBrain/Resemblyzer** - Voice embeddings

### Key Discoveries

| Finding | Details |
|---------|---------|
| **Pipeline** | Audio → VAD → ASR → Speaker Embeddings → Clustering → Output |
| **Voice Fingerprinting** | Uses x-vectors or ECAPA-TDNN embeddings (512-1920 dimensions) |
| **Mac M2 Compatible** | ✅ Whisper, Whisper.cpp, Pyannote all work on Apple Silicon |
| **Performance** | Base/medium models: real-time. Large models: slower but better quality |
| **Cost** | FREE (open source) vs $10-20/mo (Otter.ai) |

### Leading Tools
- **WhisperX** - 70x real-time, word-level timestamps, speaker labels
- **Whisper Diarization** - 5.4k GitHub stars, complete pipeline
- **Pyannote.audio** - Gold standard for speaker diarization
- **SpeechBrain** - Comprehensive speech toolkit
- **Resemblyzer** - Voice encoder for similarity

### How Otter.ai Works (Reverse-Engineered)
- Proprietary ASR + speaker embedding + clustering
- Cloud-based for speed
- Learns from user tagging
- $10-20/month subscription
