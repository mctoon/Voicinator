## 🆕 Whisper.cpp Performance Deep Dive (December 2025)

**Source:** https://sotto.to/blog/whisper-cpp-apple-silicon

**Why Whisper.cpp on Apple Silicon is Special:**

### Unified Memory Architecture
- No copying between CPU and GPU
- Model + audio data in one place
- Dramatically speeds up inference

### Optimizations:
- Core ML acceleration (Apple Neural Engine)
- Metal GPU support for M1/M2/M3
- Quantized models (smaller, faster)
- ARM NEON optimizations
- Memory-efficient inference

### Power Efficiency
- Run transcription all day without battery drain
- Consistent performance (no throttling)
