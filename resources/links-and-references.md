# Resources - Voice Fingerprinting Research

**Last Updated:** 2026-02-18

---

## GitHub Repositories

### Core ASR (Transcription)
- [OpenAI Whisper](https://github.com/openai/whisper) - 82k+ stars - Base ASR model
- [Whisper.cpp](https://github.com/ggerganov/whisper.cpp) - 35k+ stars - C++ port for Apple Silicon
- [Faster-Whisper](https://github.com/SYSTRAN/faster-whisper) - CTranslate2 backend
- [Insanely-Fast-Whisper](https://github.com/Vaibhavs10/insanely-fast-whisper) - 30x real-time

### ASR + Speaker Diarization Combined
- [WhisperX](https://github.com/m-bain/whisperX) - 15k+ stars - Recommended starting point
- [Whisper Diarization](https://github.com/MahmoudAshraf97/whisper-diarization) - 5.4k stars - Complete pipeline

### Speaker Diarization/Voice ID
- [Pyannote.audio](https://github.com/pyannote/pyannote-audio) - 5k+ stars - Gold standard
- [SpeechBrain](https://github.com/speechbrain/speechbrain) - 11k+ stars - Comprehensive toolkit
- [Resemblyzer](https://github.com/resemble-ai/Resemblyzer) - 3k+ stars - Voice encoder

### Traditional ASR
- [Kaldi](https://github.com/kaldi-asr/kaldi) - Legacy but solid, i-vector/x-vector support
- [Vosk](https://github.com/alphacep/vosk-api) - 9k+ stars - Lightweight, offline
