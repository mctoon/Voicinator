Config

Remote LLMs

Pick a provider (URLs are in llm_providers.toml), enter your API key, click Fetch models to load models from the provider, then choose a model. The LLM name is set automatically as Provider:Model. Use Test to verify the endpoint works.

Summarization parts

For each part, choose an LLM and set system prompt and user prompt. Use {{TRANSCRIPT}} in either prompt to insert the video transcript. Order here is the order in the generated summary file. Use Up/Down to reorder.
